# 《人工智能导论》实验作业 HW04
> **实验主题：** 大模型文案生成、剪映声音克隆与开源语音识别 (ASR) 实践

## 1. 项目综述
本项目完整实现了从 **“文本生成 (LLM)”** 到 **“语音合成 (TTS)”** 再到 **“语音识别 (ASR)”** 的 AI 全链路闭环。通过调用大语言模型生成创意文案，利用剪映进行个性化声音克隆，并最终使用 OpenAI Whisper 模型在本地完成语音转文字的闭环测试。
